# 🍷 Alcohol Supply Chain & Procurement Analytics
An end-to-end Data Analytics project investigating a 2016 retail procurement dataset ($321M spend).

## 🚀 Project Overview
This project identifies supply chain inefficiencies, audits pricing errors, and visualizes procurement trends to assist executive decision-making.

## 🛠️ Tech Stack
* **Python (Pandas):** Data Cleaning, Date Standardization, and Anomaly Detection.
* **Power BI:** Interactive Dashboarding and KPI tracking.
* **Jupyter Notebook:** Exploratory Data Analysis (EDA).

## 💡 Key Business Insights
* **Seasonality:** Identified major procurement peaks in **August and November**, aligned with holiday inventory loading.
* **Vendor Performance:** Benchmarked 1,000+ vendors; identified **Jim Beam** as the lead-time leader (most efficient).
* **Financial Recovery:** Discovered **$85k in missing valuation** for Brand 2520 (Macallan 12) through Brand ID cross-referencing.
